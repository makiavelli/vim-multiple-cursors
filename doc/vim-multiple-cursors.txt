vim-multiple-cursors plugin
===========================

How to play with "vim-multiple-cursors" plugin
----------------------------------------

1) First of all, press <C-F3> to power-on "vim-multiple-cursors".

Press <C-F3>, you will see something like this:

+---------------+-----+

|     		      |
+---------------------+

Legend:
	A : common text window
	B : coords window
	C : base window

2) Now save some cursor positon with <C-F4>

+---------------+-----+

|       	| 3,2 |    some positions inside "coords window"

|your face give me    |

+---------------------+

3) Enter some text in the "common text window"

				+---------------+-----+
Insert the text to write  <----	|@		| 1,3 |
inside all position saved	|       	| 3,2 |
inside "coords window" here	+---------------+-----+
				|Kitty ipsum dolor    |
				|sit amet, sleep on   |
				|your face give me    |
				|		      |
				|		      |
				|  		      |
				|     		      |
				+---------------------+

4) Press <C-F5> to write the common text inside all position

To clean the script window the users must click on <F7>, 
the editor will work in normal way.

-----

"vim-multiple-cursors" is a simple plugin that (permette) the writing of same
text inside different parts of a document.

==Mapping==
	Default mapping is enable, if you want to disable auto mapping set "let g:multiple_cursors_map_keys = 0" 
	at the end of your vimrc file.
	<C-F3> key start the plugin and the user interface
	<C-F4> key save current cursors set inside coords window
	<C-F5> key write common text(the text that must be write in different position) in
	every coord saved

== How to use "vim-multiple-cursors" ==
	The plugin works in this way:
	First of all, try to edit a simple file, than, press <C-F3>, now you will
	see three windows:
	Top right window is the coords window, when you save a cursor set, the coords
	will be saved inside this window, hereinafter referred to as 
	"coords window"
	Top left window, hereinafter referred to as 
	"common text window", is the "conatiner" of the text that you want to write starting
	from all coords saved inside coords window(the top right window).
	The thirtd window, hereinafter referred to as 
	"base window", is (sotto) the currently edited buffer
	Now, if you try to move the cursor (nella) base window and press <C-F4>, you will see
	that in the coors window verrà salvata la coordinata corrente del cursore, prova a salvare
	diverse posizioni, spostando il cursore e premendo <C-F4> per salvare le posizioni.
	Bene, a questo punto non resta che inserire del	testo da scrivere in tutte le posizioni salvate
	nella common text window e premere <C-F5>
	Fatto, ora vedrai che nelle posizioni salvate è presente il testo immesso nella common text window

== Configuration ==
At the writing time, this document there aren't many configuration available,
the only configuration flag is the "disable auto mapping" option.
If you add this at the end of your vimrc file:
	let g:multiple_cursors_map_keys = 0
then auto mapping (<C-F3> <C-F4> and <C-F5>) will be disabled

== Next steps ==
Features to be impemented inside vim-multiple-cursors:
- save coords also in the "visual mode"
- function to power-off vim-multiple-cursors plugin
- function to clear the coords window and common text window
Now "coords window" accept only singe positions, in this format:
	row,col 
Next will be (possibile) define an interval between two
coordinates, follow this two examples to (chiarire le idee):
1)	3-6,3
2)	6,12-18

In the first example "3-6,3", (è stato definito) an interval
between two rows, this means: from row 3 to row 6 and starting
from column 3 write the "common text".
In the second example "6,12-18", (è stato definito) an interval
between two column, this means: in row 6, (al posto) of the text
from column 12 to column 18 write the "common text".
